# Wheelhouse

This folder is intended for offline installation on the internal Windows machine.

It should contain wheels for:

- `bosdyn-orbit==5.1.1`
- `Flask==3.1.3`
- `waitress==3.0.2`
- `requests==2.32.5`
- `Deprecated==1.3.1`

The vendored Spot SDK wheels can be copied from `vendor/spot-sdk/prebuilt/`.
Populate the remaining third-party wheels before moving the app onto the offline machine.
